# راهنمای راه‌اندازی «لنده هوشمند» روی هاست cPanel

این بسته شامل یک وب‌سایت/اپلیکیشن کامل (بدون نیاز به Node.js یا هیچ مرحله Build) به‌علاوه یک بک‌اند PHP + MySQL است. مراحل زیر را به‌ترتیب انجام دهید.

## فایل‌های داخل این بسته
- `index.html` — کل وب‌اپلیکیشن (صفحه شهروندان + پنل مدیریت)، آماده آپلود مستقیم
- `schema.sql` — ساختار و داده‌های اولیه پایگاه داده (شامل بازارچه، اخبار، فرمانداری، شهرداری، مدیریت بحران، گردشگری، پروژه‌های عمرانی، نظرسنجی و کاربران)
- `api/` — بک‌اند PHP: `auth.php` (ورود مدیر)، `account.php` (ثبت‌نام/ورود شهروندان)، `resource.php` (اخبار/فرمانداری/شهرداری/گردشگری/رویدادها/هشدارها/پروژه‌ها)، `market.php` (بازارچه)، `reports.php` (گزارش شهروندان)، `polls.php` (نظرسنجی و رای‌گیری)

## مرحله ۱) ساخت پایگاه داده در cPanel
1. در cPanel وارد بخش **MySQL Databases** شوید.
2. یک دیتابیس جدید بسازید، مثلاً با نام `lendeh` (نام نهایی آن به‌صورت خودکار به‌شکل `cpanelusername_lendeh` خواهد بود).
3. یک کاربر MySQL جدید بسازید و رمز عبور قوی برایش انتخاب کنید.
4. کاربر را به دیتابیس اضافه کنید و دسترسی **All Privileges** را به او بدهید.
5. نام دیتابیس، نام کاربری و رمز عبور را یادداشت کنید.

## مرحله ۲) ایمپورت ساختار پایگاه داده
1. از cPanel وارد **phpMyAdmin** شوید.
2. دیتابیسی که ساختید را از سمت راست انتخاب کنید.
3. به تب **Import** بروید، فایل `schema.sql` را انتخاب و Import کنید.
4. باید جدول‌های news، market، government، municipality، crisis_alerts، crisis_reports، tourism، events، projects، polls، poll_votes و users همراه با چند رکورد نمونه ساخته شده باشند.

## مرحله ۳) تنظیم اطلاعات اتصال
فایل `api/config.php` را با ویرایشگر متنی باز کنید و این مقادیر را با اطلاعات مرحله ۱ جایگزین کنید:

```php
define('DB_HOST', 'localhost');
define('DB_NAME', 'cpanelusername_lendeh');
define('DB_USER', 'cpanelusername_lendeh');
define('DB_PASS', 'رمز-عبور-شما');
```

## مرحله ۴) آپلود فایل‌ها
1. وارد **File Manager** در cPanel شوید و به پوشه `public_html` بروید (یا پوشه دامنه/ساب‌دامنه موردنظر).
2. فایل `index.html` را مستقیماً داخل `public_html` آپلود کنید.
3. پوشه کامل `api` (با همه فایل‌های داخلش) را هم داخل `public_html` آپلود کنید، طوری که آدرس نهایی به‌صورت `public_html/api/config.php` و غیره باشد.
4. ساختار نهایی باید این‌گونه باشد:
   ```
   public_html/
     index.html
     api/
       config.php
       db.php
       helpers.php
       auth.php
       resource.php
       market.php
       reports.php
       generate_password.php
   ```

> نکته: راحت‌ترین روش این است که ابتدا این پوشه را به‌صورت ZIP فشرده کنید، آن را در File Manager آپلود کنید و سپس با گزینه **Extract** داخل `public_html` از حالت فشرده خارج کنید.

## مرحله ۵) تست سایت
1. آدرس دامنه‌تان را در مرورگر باز کنید؛ باید صفحه اصلی «لنده هوشمند» نمایش داده شود.
2. از پایین صفحه روی «ورود مدیر سامانه» بزنید و با اطلاعات پیش‌فرض وارد شوید:
   - نام کاربری: `admin`
   - رمز عبور: `lendeh1404`
3. از پنل مدیریت یک خبر یا آگهی آزمایشی ثبت کنید و مطمئن شوید در سایت اصلی نمایش داده می‌شود.

## مرحله ۶) تغییر رمز عبور مدیر (ضروری)
رمز پیش‌فرض را حتماً تغییر دهید:
1. آدرس `https://دامنه‌شما.ir/api/generate_password.php?password=رمز-جدید-دلخواه-من` را در مرورگر باز کنید.
2. رشته هش‌شده‌ای که نمایش داده می‌شود را کپی کنید.
3. در فایل `api/config.php`، مقدار `ADMIN_PASSWORD_HASH` را با همین رشته جایگزین کنید.
4. فایل `api/generate_password.php` را از روی هاست **حذف کنید** (این فایل فقط برای یک‌بار استفاده است).

## نکات امنیتی و فنی
- حتماً **SSL رایگان (AutoSSL / Let's Encrypt)** را از بخش SSL/TLS Status در cPanel برای دامنه فعال کنید تا سایت روی HTTPS بالا بیاید.
- این نسخه از React و Tailwind از طریق CDN استفاده می‌کند (بدون نیاز به Node.js)، که برای راه‌اندازی سریع روی هاست اشتراکی مناسب است. اگر در آینده ترافیک سایت بالا رفت یا خواستید سرعت بارگذاری را به حداکثر برسانید، می‌توان نسخه‌ی build‌شده (Vite) را جایگزین کرد؛ کافی است درخواست بدهید.
- شماره‌های تماس اضطراری داخل برنامه (پلیس، اورژانس، آتش‌نشانی و...) شماره‌های سراسری کشور هستند؛ شماره دفاتر محلی فرمانداری/شهرداری لنده را از طریق پنل مدیریت یا ویرایش مستقیم کد اضافه کنید.
- تمام محتوای اولیه (اخبار، آگهی‌ها، جاذبه‌های گردشگری و...) صرفاً نمونه است و باید از طریق پنل مدیریت با اطلاعات واقعی جایگزین یا تکمیل شود.

## درباره بخش‌های جدید
- **رای‌گیری/نظرسنجی**: برای جلوگیری از رای تکراری، رای دادن نیازمند ورود با حساب کاربری (ایمیل) است. هر کاربر فقط یک بار می‌تواند به هر نظرسنجی رای دهد. مدیریت نظرسنجی‌ها از بخش «نظرسنجی و رای‌گیری» در پنل مدیریت انجام می‌شود.
- **اسلاید اخبار صفحه اول**: به‌صورت خودکار از ۵ خبر آخر ساخته می‌شود و نیازی به تنظیم جداگانه ندارد؛ صرفاً اخبار را از پنل مدیریت ثبت کنید.
- **پروژه‌های عمرانی**: هر پروژه شامل درصد پیشرفت (نوار پیشرفت تصویری)، نیازمندی‌ها و موانع پیش‌رو است. مدیریت کامل از بخش «پروژه‌های عمرانی» در پنل امکان‌پذیر است.
- **ثبت‌نام کاربران با ایمیل**: شهروندان از طریق «بیشتر ← ورود/ثبت‌نام» با نام، ایمیل و رمز عبور ثبت‌نام می‌کنند. فهرست کاربران ثبت‌نامی از بخش «کاربران ثبت‌نام‌شده» در پنل مدیریت قابل مشاهده و حذف است.

### نکته مهم درباره ارسال ایمیل
سامانه هنگام ثبت‌نام یک ایمیل خوش‌آمدگویی ساده با تابع داخلی `mail()` در PHP ارسال می‌کند. این روش روی اکثر هاست‌های cPanel کار می‌کند، اما:
- ممکن است ایمیل‌ها در پوشه Spam گیرنده قرار بگیرند (چون از SMTP احراز‌هویت‌شده استفاده نمی‌شود).
- مقدار `MAIL_FROM` در فایل `api/config.php` را به یک ایمیل واقعی روی همان دامنه (مثلاً `no-reply@دامنه‌شما.ir`) تغییر دهید تا نرخ تحویل ایمیل بهتر شود.
- ثبت‌نام و ورود کاربران به ارسال موفق ایمیل وابسته نیست؛ یعنی حتی اگر ارسال ایمیل با خطا مواجه شود، ثبت‌نام کاربر همچنان با موفقیت انجام می‌شود.
- برای تحویل‌پذیری حرفه‌ای ایمیل (مثلاً تایید ایمیل با لینک)، توصیه می‌شود بعداً از یک سرویس SMTP مثل Mailgun یا SendGrid استفاده شود؛ در صورت نیاز می‌توانم این بخش را هم اضافه کنم.

## مشکلی پیش آمد؟
- صفحه سفید یا خطای JSON: یعنی اتصال به دیتابیس برقرار نشده — `api/config.php` را دوباره بررسی کنید.
- ورود به پنل مدیریت کار نمی‌کند: مطمئن شوید دامنه‌تان روی HTTPS است (مرورگرها کوکی نشست را در HTTP محدود می‌کنند).
